archive #1
FragFest
image reference
when
2014
thoughts
unlimited movement, overpowered railguns, hud speedometers, this was my first project in an era where i started exploring gamedev as a real possibility, a "half-life 2 deathmatch" mod that served as a playground for learning source engine programming.

this mod essentially prepared me for pvk2 (the first game on the page), and was my first dive into c++ programming as a whole, the gameplay itself is bonkers and surprisingly fun - think a "babby's first mod" trying to pass as a proper quake ripoff lmao!

the mod was codenamed "the nab project", which was a reference to a group of friends at the time.

archive #2
OVERHEAT
image reference
when
2019
thoughts
overheat was an ambitious solo game project that started out as a joke to test my 2d game engine
(called YASE, which stands for "Yet Another Sh*t Engine", on good days, S meant "SDL")

it was to be a kusoge-inspired shmup w/ bold colorful visuals. quite a bit of 'promo' material and a full soundtrack was made, but the game itself was never realized beyond a very crude tech demo that isn't worth checking out.

this is one of the projects i *may* return to in a different capacity, at least the stylistic choice with rave-infused music and visual language - but hey, who knows?
